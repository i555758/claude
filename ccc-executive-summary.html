CCC Executive Summary

Claude Code command that generates an executive summary dashboard for your entire cloud project portfolio — live from the SAP Cloud Reporting portal.

Prerequisites: Claude Code CLI installed  •  SAP MCP Suite configured in ~/.claude/settings.json  •  Access to reporting.ondemand.com

What this command does

  • Fetches live data from the SAP Cloud Reporting portal (your active project portfolio)
  • Scans recent emails via Microsoft Graph API (last 3 days) for project signals
  • Generates per-project summaries: Executive Summary (max 500 chars) + Simple Explanation (1–2 sentences)
  • Saves an interactive HTML dashboard to C:\Users\{YOU}\ExecutiveSummary\executive_summary_latest.html
  • Opens the dashboard automatically — includes checkboxes, filter tabs, and review progress bar

How to install

1
Click "Download Installer" below and save the zip to your Desktop.
2
Extract the zip, then double-click install-ccc-executive-summary.cmd. No admin required.
3
The installer copies CCC-Executive-Summary.md to %USERPROFILE%\.claude\commands\ and confirms success.
4
Open Claude Code and type /CCC-Executive-Summary to run the command.
