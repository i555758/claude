Claude Code Launcher

Install the Claude Code Launcher on your Windows notebook in two steps.

Prerequisites: Claude Code CLI installed  •  HAI Proxy installed at C:\SAPDevelop\hai\hai.exe  •  SAP MCP Suite configured in ~/.claude/settings.json

How to install

1
Click "Download Installer" below and save the zip to your Desktop.
2
Extract the zip, then double-click instalar-claude-code.cmd. No admin required.
3
The installer copies the files to C:\SAPDevelop\claude-launcher\ and creates a Claude Code shortcut on your Desktop with the official icon.
4
Use the Desktop shortcut to launch Claude Code. The HAI Proxy starts automatically and Claude opens when the proxy is ready.
